# Instagram Homepage Clone

This is a **responsive front-end clone** of Instagram’s homepage interface built using **HTML**, **CSS**, and **Font Awesome icons**. 
It replicates the basic layout and user interface of Instagram, including the header, stories, posts, and suggestions panel.

---

## 🔧 Technologies Used

- ✅ HTML5
- ✅ CSS3
- ✅ Font Awesome (for icons)
- ✅ Responsive Design principles

---

## 📁 Folder Structure
instagram-clone/
├── index.html
├── insta.css
├── images/
│ └── (All image assets)
├── README.md


---

## 📌 Features

- Instagram-style header with logo, search bar, and navigation icons.
- Story section displaying user avatars.
- Post cards with profile image, post image, action icons (like, comment, share, save).
- Comments and likes section.
- Suggestions sidebar with follow options.
